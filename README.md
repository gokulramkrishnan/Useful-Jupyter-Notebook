The purpose of the interactive python notebooks is to provide predefined use cases and their executable codes that is compatible Python, Databricks and other Python IDEs. All the packages used as part of the notebooks are open source, and at the time of the code commits was extensible to Commercial use. But the users of the notes are suggested to review the licensing details for commercializig and for enterprise deployment needs. This repository is a consolidation of different use cases, concepts and logics as a quick reference for industry use.
